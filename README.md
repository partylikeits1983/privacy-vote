## Privacy Vote 

This repository is a basic demonstration of the Noir zk-circuit language, the WebAuthn API, and a Solidity Smart contract to enable a KYC'ed private voting mechanism.


### Running 
```
pnpm i
pnpm build 
```

```
pnpm start:zkp
```

In a new terminal:
```
pnpm deploy:contracts
pnpm start
```
